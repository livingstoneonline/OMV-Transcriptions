Image Prep Workflow

0) Use Zotero to set item name

1) Save original image as TIF with UID name: liv_012001_0001.tif

2) Convert in terminal

convert liv_012001_0001.tif -resize 1500x1500 liv_012001_0001-1500px.jpg &&
convert liv_012001_0001.tif -resize 900x900 liv_012001_0001-900px.jpg && 
convert liv_012001_0001.tif -resize 750x750 liv_012001_0001-750px.jpg && 
convert liv_012001_0001.tif -resize 600x600 liv_012001_0001-600px.jpg && 
convert liv_012001_0001.tif -resize 450x450 liv_012001_0001-450px.jpg

or 

convert liv_012001_0001.jpg -resize 1500x1500 liv_012001_0001-1500px.jpg &&
convert liv_012001_0001.jpg -resize 900x900 liv_012001_0001-900px.jpg && 
convert liv_012001_0001.jpg -resize 750x750 liv_012001_0001-750px.jpg && 
convert liv_012001_0001.jpg -resize 600x600 liv_012001_0001-600px.jpg && 
convert liv_012001_0001.jpg -resize 450x450 liv_012001_0001-450px.jpg

3) Add code to HTML

class="image" src="site-images/liv_012001_0001-750px.jpg" srcset="site-images/liv_012001_0001-450px.jpg 450w, site-images/liv_012001_0001-600px.jpg 600w, site-images/liv_012001_0001-750px.jpg 750w, site-images/liv_012001_0001-900px.jpg 900w" sizes="(min-width: 1000px) 638px, (min-width: 780px) 699px, calc(100vw - 32px)"

[Note that the 1500 image isn't included.]

4) Add images to relevant folder

5) Open page and calculate image sizes using https://ausi.github.io/respimagelint/ and correct as needed.
